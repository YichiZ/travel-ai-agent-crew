# AI Travel Planner Frontend

A React-based frontend application for the AI-powered travel planning system. This application allows users to search for flights and hotels, and receive AI-generated itineraries.

## Features

- **Smart Travel Search**: Search for flights and hotels in one go
- **AI-Powered Recommendations**: Get intelligent recommendations for both flights and hotels
- **Personalized Itineraries**: Receive custom travel itineraries generated by AI
- **Beautiful UI**: Modern, responsive design with smooth animations
- **Real-time Updates**: Loading states and error handling for a great user experience

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Backend API running (default: http://localhost:8000)

## Installation

1. Install dependencies:
```bash
npm install
```

2. Configure the backend API URL:
   - Copy `.env.example` to `.env`
   - Update `VITE_API_URL` if your backend is running on a different URL

## Running the Application

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173` (default Vite port).

## Building for Production

Create a production build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## API Integration

The frontend communicates with the backend via the `/complete_search/` endpoint:

**Request Format:**
```json
{
  "flight_request": {
    "origin": "SFO",
    "destination": "JFK",
    "outbound_date": "2025-12-01",
    "return_date": "2025-12-15"
  },
  "hotel_request": {
    "location": "New York",
    "check_in_date": "2025-12-01",
    "check_out_date": "2025-12-15"
  }
}
```

**Response Format:**
```json
{
  "flights": [...],
  "hotels": [...],
  "ai_flight_recommendation": "...",
  "ai_hotel_recommendation": "...",
  "itinerary": "..."
}
```
